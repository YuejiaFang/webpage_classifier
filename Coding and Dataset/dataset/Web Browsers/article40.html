Troubleshooting: Web browsers

A Web browser is a program that you use to view web pages on the Internet from your computer or mobile device. Having issues navigating the Web? Use these tips to address issues with your web browser (eg. Chrome, Safari, Microsoft Edge, Firefox).

Troubleshooting: Quick Fixes

There are three things you can do to try and resolve your Web browser issues:

  1. Update your browser to the latest available version. Having the latest version of your browser also makes your browsing experience safer. See the Related Reading below for instructions on how to update your Web browser.
  2. Clear your browser's history (of all previously viewed files and pages).
  3. Reset your browser.  Note that this will erase everything, including personal settings and saved passwords.
You can usually find these options in your browser's menu bar under Settings or Preferences. If you need help with your specific web browser, visit the developer's support website.
